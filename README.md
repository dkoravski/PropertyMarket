# PropertyMarket

PropertyMarket is a multi-page web application for buying, selling and renting residential properties (apartments, houses, villas, guest houses, etc.).

## Project Overview

The platform allows users to:
- register and login;
- create, edit and delete property listings;
- browse all published properties;
- save properties to favorites;
- manage their profile;
- access admin features (for admin users).

Site visitors (including non-authenticated users) can view and browse all published property listings.

## Tech Stack

- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Supabase (PostgreSQL, Auth, Storage)
- Build tools: Node.js, npm, Vite
- Hosting: Netlify
- Source Control: GitHub
